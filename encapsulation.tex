\section{Encapsulation}

\subsection{Link Layer VPNs (Layer 2)}
Beispiele für VPNS auf dieser Ebene sind Integrated Services Digital Network (ISDN), Frame Realy und Asynchronous Transfer Mode. Auch Virtual Local Networks funktionieren auf ähnliche Weise auf dieser Ebene.

\paragraph{Vorteile}
\begin{itemize}
\item Verbindungsorientiert, Pakete müssen nicht geroutet werden sondern werden über einmal erstellten Link übertragen.
\item QoS auf dieser Ebene schon garantiert, muss nicht noch zusätzlich implementiert werden.
\end{itemize}

\paragraph{Nachteile}
\begin{itemize}
\item Braucht homogene Netzwerkstruktur
\item IP-Ebene muss trotzdem noch verwaltet werden, Mehraufwand auf zwei Ebenen!
\end{itemize}
